# Aircraft Performance Calculator

## Project Overview

This project is developed using Python to calculate basic aircraft performance parameters such as flight time, fuel consumption, remaining fuel, and mission feasibility.

The project uses the HAL Tejas Mk1A as the reference aircraft.
